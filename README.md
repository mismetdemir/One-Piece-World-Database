# One-Piece-World-Database


## Project Description

One Piece World Database is a web-based CRUD application that allows users to manage character information from the One Piece universe.

Users can create, read, update, and delete character records. The system also supports searching characters by name, filtering them by crew, and sorting them by bounty.

## Main Entity

### Character

The main entity of the system is **Character**. Each character record represents a character from the One Piece universe.

## Main Features

- Add new character
- View all characters
- Update character information
- Delete character
- Search character by name
- Filter characters by crew
- Sort characters by bounty

## Project Purpose

The purpose of this project is to create a simple and functional CRUD-based database system for managing One Piece character information. It is designed as a Systems Analysis and Design project and focuses on basic data management operations, filtering, searching, sorting, API documentation, and testing.

## CRUD Operations

The system supports the following CRUD operations:

| Operation | Description |
|---|---|
| Create | Add a new character record |
| Read | View all character records |
| Update | Edit existing character information |
| Delete | Remove a character record |

## Additional Functionalities

In addition to basic CRUD operations, the system includes:

- Character search by name
- Crew-based filtering
- Sorting characters by bounty amount
